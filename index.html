Mini Project · Embedded Systems
Human Centric Embodied Home Assistant

A mobile smart home companion designed to facilitate natural human-machine interactions. The system operates on a Raspberry Pi orchestrating core cognitive tasks—like local facial recognition via OpenCV and natural speech responses—while communicating with distributed ESP32 nodes. When radar sensors detect proximity, a localized OLED screen renders dynamic digital eyes, displaying expressive feedback to emulate personality.

Micro Project · Arduino
Automated Rain Shield System

This project focuses on automated structural protection. Built around an Arduino microcontroller, the system employs rain sensor arrays to monitor moisture levels continuously. Upon detecting precipitation above a calibrated threshold, the controller outputs PWM signals to a high-torque servo motor, deploying a folding waterproof canopy. The real-time deployment status, humidity levels, and system diagnostics are printed directly to a local LCD interface.

VR Workshop · Unity
Virtual Horizon

Virtual Horizon is an immersive VR platform co-designed to showcase educational and digital tourism application designs. Using Unity, we built interactive virtual spaces that allow users to inspect historical models and navigate virtual environments. Custom components designed in Fusion 360 were imported, textured, and integrated using C# scripting to drive physical actions and object manipulation in virtual space.
