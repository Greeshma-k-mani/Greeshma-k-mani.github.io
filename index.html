Mini Project · Embedded Systems
Human Centric Embodied Home Assistant
A Raspberry Pi-based smart home assistant with 3DOF wheel-based mobility, voice interaction, vision-based face recognition, and radar-triggered OLED facial feedback running across ESP32 and ESP8266 hardware. The robot can navigate autonomously, recognize family members by face, respond to voice commands, and display emotional feedback on its OLED screen based on proximity detected by radar sensors.

An Arduino-based automated rain protection system using sensor-driven decision-making to trigger a motorized shield mechanism, with real-time monitoring and display output.
